自己搭的一个 vue 的开发模版，欢迎 issue 和 pr 来交流

**已经实现：**

- 基础的 vue，vuex，vue-router，
- webpack 打包
- vue-router 使用 hash 模式
- 添加了 eslint+prettier 控制代码质量
- 添加了 git 钩子函数处理

## 使用方法

1. 直接下载使用

```js
git clone https://github.com/kaisa911/vue-template.git

cd vue-template

npm i
```

2. 使用 Paji-Cli 生成

```js
npm i paji-cli -g

cd /dir

PajiCli create my-App
```
